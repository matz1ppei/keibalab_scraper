# keibalab_scraper
crawling & scraping [Keibalab](https://www.keibalab.jp/)

## Usage
### Initialize
```
pipenv install
```
### Crawling
```
python spider/keibalab_spider.py <from_date> <to_date>
```

### Scraping
```
python scraper/keibalab_scraper.py

for MariaDB
python scraper/keibalab_scraper_for_mariadb.py
```
