# keibalab_scraper
crawlering & scraping [Keibalab](https://www.keibalab.jp/)

## Usage
### Initialize
```
pipenv install
```
### Crawling
```
python spider/keibalab_spider.py <from_date> <to_date>
```
$python spider/keibalab_spider.py 20191201 20191231

### Scraping
```
python scraper/keibalab_scraper.py
```
